# Magento 2 Hack Competitor Extension by [Magefan](https://magefan.com)

Few people know that almost every website in the world has a vulnerability that can be exploited once a year and only on April 1.
Using this simple Magento 2 extension you can hack your competitor's website and get the business advantage.

<a href="https://github.com/magefan/module-hack-competitor/archive/refs/heads/main.zip"><img width="300" height="60" src="https://cm.magefan.com/mf_webp/png/media/wysiwyg/DOWNLOAD_NOW.webp"></a>

## Requirements
  * Magento Community 2.3.0-2.4.x (CE, EE, ECE, B2B)


## How to Install?

Run these CLI commands in Magento 2 root directory:
````
wget -O magefan-hack-competitor.zip https://github.com/magefan/module-hack-competitor/archive/refs/heads/main.zip
mkdir -p app/code/Magefan
unzip magefan-hack-competitor.zip -d app/code/Magefan/
mv app/code/Magefan/module-hack-competitor-main app/code/Magefan/HackCompetitor
rm magefan-hack-competitor.zip
bin/magento setup:upgrade --keep-generated
bin/magento setup:di:compile
````

## How to Use?

Run this CLI command in Magento 2 root directory:
```
bin/magento magefan:hack:competitor google.com
```
